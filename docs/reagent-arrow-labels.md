# Reagent Arrow Labels

These are the current canonical labels chemrulez should show in reagent chips,
path steps, and copied routes. They are meant to be close to what a student would
write over the reaction arrow on an exam.

| Transform | Arrow label |
| --- | --- |
| Terminal alkyne deprotonation | `NaNH2` |
| Catalytic hydrogenation | `H2, Pd/C` |
| Lindlar partial alkyne reduction | `H2, Lindlar` |
| Dissolving-metal alkyne reduction | `Na, NH3(l)` |
| Alkyne mercuric hydration | `HgSO4, H2SO4, H2O` |
| Terminal alkyne hydroboration-oxidation | `1. (sia)2BH  2. H2O2, NaOH` |
| Hydrohalogenation | `HBr` |
| Radical anti-Markovnikov hydrobromination | `HBr, ROOR` |
| Acid-catalyzed alkene hydration | `H2SO4, H2O` |
| Hydroxide substitution/basic epoxide opening | `NaOH` |
| Cyanide substitution | `NaCN` |
| E2 with small base | `NaOEt, EtOH, heat` |
| E2 with bulky base | `t-BuOK, t-BuOH, heat` |
| E1 solvolysis | `EtOH, heat` |
| Oxymercuration-demercuration | `1. Hg(OAc)2, H2O  2. NaBH4` |
| Alkene hydroboration-oxidation | `1. BH3, THF  2. H2O2, NaOH` |
| Alkene bromination | `Br2, CCl4` |
| Alkene epoxidation | `mCPBA` |
| Syn dihydroxylation | `1. OsO4  2. NaHSO3, H2O` |
| Reductive ozonolysis | `1. O3  2. DMS` |
| Grignard formation | `Mg, Et2O` |
| Alcohol tosylation | `TsCl, pyridine` |
| Alcohol to alkyl bromide | `PBr3` |
| Alcohol to alkyl chloride | `SOCl2, pyridine` |
| PCC oxidation | `PCC, CH2Cl2` |
| Dess-Martin oxidation | `DMP, CH2Cl2` |
| Swern oxidation | `1. (COCl)2, DMSO  2. Et3N` |
| Jones/dichromate oxidation | `Na2Cr2O7, H2SO4, H2O` |
| Hot acidic permanganate oxidation | `KMnO4, H3O+, heat` |
| Sodium borohydride carbonyl reduction | `1. NaBH4  2. H3O+` |
| Lithium aluminum hydride reduction | `1. LiAlH4  2. H3O+` |
| DIBAL-H ester reduction | `1. DIBAL-H, toluene, -78 C  2. H3O+` |
| Friedel-Crafts acylation conditions | `AlCl3` plus an acid chloride co-reagent |
| Grignard addition | `RMgX, H3O+` or a specific `RMgBr, H3O+` |

Open cleanup items:

- Some labels still encode one common acceptable exam answer rather than every
  valid variant. For example, `Br2, CCl4` can also be written with another inert
  solvent.
- Reagent aliases may be broad, but canonical labels should stay specific. If
  an alias implies a different exam arrow, split it into its own reagent entry.
- Reaction rules should use graph transforms. This file is display/provenance
  metadata, not a product-generation mechanism.
