# Textbook Synthesis Cases

These are good smoke tests for the first-year organic synthesis sandbox.

- Build an internal alkyne: `1-butyne` -> `NaNH2` -> `ethyl bromide`.
- Partial alkyne reductions: `2-butyne` -> `H2, Lindlar` and `2-butyne` -> `Na, NH3`.
- Stereochemical alkyne reductions: `2-butyne` -> `Na, NH3 (l)` should show the trans alkene; `2-butyne` -> `H2, Lindlar` should show the cis alkene.
- Full hydrogenation: `propene` or `2-butyne` -> `H2, Pd/C`.
- Terminal alkyne hydration contrast: `1-butyne` -> `HgSO4, H2SO4, H2O` versus `1-butyne` -> `Sia2BH then H2O2 NaOH`.
- Alkene regiochemistry contrast: `propene` -> `HBr` versus `propene` -> `HBr, ROOR`.
- Branched HBr addition: `methyl 2-butene` or `2-methyl-2-butene` -> `HBr` should give the tertiary bromide.
- Carbocation rearrangement: `3-methyl-1-butene` -> `HBr` should rank 2-bromo-2-methylbutane as the major rearranged product and still show the unrearranged option.
- Methyl-shift rearrangement: `3,3-dimethyl-1-butene` -> `HBr` should rank the methyl-shift tertiary bromide as major.
- No alkene, no HBr addition: literal `3,3-dimethylbutane` -> `HBr` should not produce hydrohalogenation candidates.
- Alcohol regiochemistry contrast: `propene` -> `Hg(OAc)2, H2O then NaBH4` versus `propene` -> `BH3 then H2O2 NaOH`.
- Alkene cleavage: `propene` -> `O3 then DMS` should give acetaldehyde plus formaldehyde.
- Grignard carbonyl addition: `acetaldehyde` -> `CH3MgBr then H3O+` should give a secondary alcohol.
- Grignard carboxylation: `CO2` -> `PhMgBr then H3O+` should give benzoic acid.
- Stereochemistry families to track later: `2-butene` -> `Br2`, `OsO4`, `mCPBA`.
- Carbocation rearrangement fan-out: `3-methyl-1-butene` -> `H3O+` or `HBr`.
- Bad acetylide SN2 partner: acetylide plus `tert-butyl bromide` should block and explain elimination.
- Secondary alkyl halide warning: acetylide plus `2-bromopropane` should show competing SN2/E2.
